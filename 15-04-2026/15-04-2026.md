## Xây dựng mô hình LEMP
- LEMP stack: cần có linux, nginx, mysql, php để hoạt động
  + Cài đặt nginx: `apt install nginx`
  <img width="821" height="486" alt="image" src="https://github.com/user-attachments/assets/f7919f76-126d-4749-af89-7952a3a3beb0" />

  + Kiểm tra service nginx: `systemctl status nginx`
  <img width="814" height="421" alt="image" src="https://github.com/user-attachments/assets/3504eb46-83c2-4ec2-8011-fda2e5fa6ce1" />
  
  + Kiểm tra xem máy chủ web đang chạy bằng cách truy cập IP của máy chủ
  <img width="1286" height="415" alt="image" src="https://github.com/user-attachments/assets/fcea8ec9-7f9a-4426-a814-cf0e7b616689" />

  + Cài đặt mysql server: `apt install mysql-server -y`
  <img width="814" height="349" alt="image" src="https://github.com/user-attachments/assets/b40bc302-d4fe-46bd-b200-5ccc7abb4662" />

  + Kiểm tra service mysql: `systemctl status mysql`
  <img width="896" height="352" alt="image" src="https://github.com/user-attachments/assets/f028d3b4-1ad5-4d13-a1f8-d771300caf47" />

  + Cài đặt php8.1: `sudo apt install -y php8.1-*`, ký tự `*` thể hiện cho tất cả extention của php được cài thay vì nhập nhiều extention trong command install. Hoặc dùng command `apt install php8.1-fpm php-mysql` để vừa đủ extention cần thiết.
  <img width="1636" height="430" alt="image" src="https://github.com/user-attachments/assets/ff1c300b-5e5d-4273-aaf1-f406ee5eafee" />

  + Kiểm tra service php: `systemctl status php8.1-fpm`

Reverse proxy: là máy chủ trung gian nằm phía trước các máy chủ backend (web server, ứng dụng), nhận yêu cầu từ internet và chuyển tiếp đến các server phù hợp. Nó giúp cân bằng tải, bảo mật, và ẩn địa chỉ IP của server gốc, giúp tăng cường bảo mật và hiệu suất toàn bộ hệ thống.
## Apache
  - Cài đặt apache: `apt install apache2 libapache2-mod-php8.1 -y`
  - Sửa cấu hình file `nano /etc/apache2/ports.conf` tìm `Listen 80` đổi 80 thành 8080 vì nginx đang dùng
  - Tương tự với `nano /etc/apache2/sites-available/000-default.conf` tìm `VirtualHost *:80` đổi 80 thành 8080
  - Khởi động lại apache2 `systemctl restart apache2`

    ```
    #Kích hoạt module rewrite
    a2enmod rewrite
    
    # Kích hoạt 2 file cấu hình
    a2ensite wordpress.conf
    a2ensite laravel.conf

    # Tắt file default để tránh xung đột
    a2dissite 000-default.conf

    # Kiểm tra trước khi khởi động lại
    apache2ctl configtest

    # Nếu Syntax OK
    systemctl reload apache2
    ```

Nginx đứng trước Apache -> Nginx đóng vai trò là reverse proxy, để kết nối, tiếp nhận file tĩnh (nginx xử lý file ảnh/css nhanh hơn apache) và xử lý SSL để Apache xử lý các file động. 

## Apache
  - Cài đặt apache: `apt install apache2 libapache2-mod-php8.1 -y`
  - Sửa cấu hình file `nano /etc/apache2/ports.conf` tìm `Listen 80` đổi 80 thành 8080 vì nginx đang dùng cổng 80, thêm `Listen 8443` cho https
  - Khởi động lại apache2 `systemctl restart apache2`

server {
    listen 80;
    server_name wp.giahung.vietnix.tech;
    root /var/www/giahung/wordpress;

    location ~* \.(jpg|jpeg|png|gif|ico|css|js|woff|woff2|ttf|svg)$ {
        expires 30d;
        try_files $uri =404;
        access_log off;
    }

    location / {
        proxy_pass http://127.0.0.1:8080;
        include proxy_params;
    }

    set $redirect_https 0;
    if ($redirect_https = 1) {
        return 301 https://$host$request_uri;
    }

}
